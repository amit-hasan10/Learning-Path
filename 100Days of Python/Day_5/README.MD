# Python Variables and Data Types

## Overview
Variables are used to store data, and data types define what kind of data a variable holds.
In Python, every value has a data type, and the type determines what operations can be performed on that value.

---

## Variables in Python

A **variable** is a name that refers to a value stored in memory.

### Example:
```python
name = "Marua"
age = 21
height = 5.6
is_student = True

print(name)
print(age)
print(height)
print(is_student)
