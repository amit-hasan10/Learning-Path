ğŸ”¤ Changing Case
lower()

Converts all characters to lowercase.

"Hello World".lower()   # 'hello world'

upper()

Converts all characters to uppercase.

"Hello World".upper()   # 'HELLO WORLD'

title()

Capitalizes first letter of each word.

"python programming".title()   # 'Python Programming'

capitalize()

Capitalizes only the first character.

"python".capitalize()   # 'Python'

ğŸ” Searching in Strings
find(substring)

Returns index of first occurrence (-1 if not found):

"python".find("th")   # 2

rfind(substring)

Searches from the right:

"banana".rfind("a")   # 5

index() / rindex()

Like find() but throws error if not found:

"hello".index("e")   # 1

ğŸ”„ Replacing Text
replace(old, new)
"Java is fun".replace("Java", "Python")
# 'Python is fun'

âœ‚ï¸ Splitting & Joining
split(separator)
"a,b,c".split(",")   # ['a', 'b', 'c']

join(iterable)
"-".join(["2025", "01", "01"])
# '2025-01-01'

ğŸ§¹ Removing Whitespace
strip()

Removes whitespace from both ends.

"  hello  ".strip()   # 'hello'

lstrip()
"   hi".lstrip()   # 'hi'

rstrip()
"bye   ".rstrip()   # 'bye'

ğŸ”¢ String Check Methods

Return True/False.

isdigit()
"123".isdigit()   # True

isalpha()
"abc".isalpha()   # True

isalnum()
"abc123".isalnum()   # True

islower() / isupper()
"hello".islower()   # True

isspace()
"   ".isspace()   # True

ğŸ¨ Formatting Methods
center(width)

# '    hi    '

# 'Amit------'

rjust(width, fillchar)
"Amit".rjust(10, ".")
# '......Amit'
